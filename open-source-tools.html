Open-Source Vulnerability Scanners

OWASP ZAP (Zed Attack Proxy)

One of the most popular open-source web application security scanners maintained by OWASP. It allows you to find vulnerabilities in web applications.

Key Features: Active and passive vulnerability scanning, API integrations for DevOps pipelines, automated scans.

Visit OWASP ZAP GitHub

OpenVAS

OpenVAS is a comprehensive open-source network vulnerability scanner. It includes thousands of pre-configured vulnerability tests.

Key Features: Network vulnerability scanning, thousands of vulnerability tests.

Visit OpenVAS GitHub

Nmap

Nmap is a free and open-source network discovery and security auditing tool. It’s widely used to discover hosts, services, open ports, and vulnerabilities.

Key Features: Network mapping, port scanning, OS fingerprinting, vulnerability detection.

Visit Nmap Official Website

Wapiti

Wapiti is a web vulnerability scanner that audits the security of web applications. It detects vulnerabilities like SQL injection, XSS, and file inclusions.

Key Features: Web application vulnerability scanning, support for various vulnerability types.

Visit Wapiti GitHub

Nikto

Nikto is a free and open-source web server scanner that performs comprehensive tests against web servers to find potential vulnerabilities.

Key Features: Detection of dangerous files, outdated software, misconfigurations, etc.

Visit Nikto GitHub

Metasploit Framework

The Metasploit Framework is an open-source penetration testing tool that helps you exploit known vulnerabilities. It’s a powerful tool used by both attackers and defenders.

Key Features: Exploitation of vulnerabilities, penetration testing automation.

Visit Metasploit GitHub

Clair

Clair is an open-source project used for static analysis of vulnerabilities in containerized applications like Docker.

Key Features: Detect vulnerabilities in container images, integrate with container registries.

Visit Clair GitHub
